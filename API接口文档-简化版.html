AI 人才评估 — API 接口文档（简化版）

版本号：v1.1.0  |  更新日期：2026-07-21

1. 文档概述

本文档描述 AI 人才评估项目的核心 RESTful API，用于上传新人主播的面试视频及照片，并查询 Google Gemini 大模型对 10 大类 20 小项能力的综合评估结果。

2. 认证方式

所有接口均需在 HTTP Header 中携带 Authorization：

Authorization: Bearer <authKey>

authKey 由系统管理员统一分配，未携带或无效将返回 401。

3. 通用响应格式

{
  "code": 0,
  "message": "success",
  "data": {},
  "timestamp": 1752979200000
}
字段类型说明
codeinteger0 成功，非 0 异常
messagestring状态描述
dataobject / null业务数据，失败时为 null
timestampinteger (int64)服务端响应时刻的 Unix 毫秒时间戳

4. 接口一：POST 上传视频/图片

  • URL：/evaluation/upload
  • Method：POST
  • Content-Type：multipart/form-data

请求参数

参数名类型必填说明
filesFile[]是视频和图片文件。视频(.mp4/.mov/.avi)，图片(.jpg/.jpeg/.png/.webp)。至少 1 个，最多 10 个
candidateNamestring是候选人姓名，1–50 个字符
candidateIdstring否候选人编号
remarkstring否备注信息，最多 500 个字符
callbackUrlstring否评估完成后的回调地址
文件限制：视频单文件 ≤ 500MB，图片单文件 ≤ 20MB

curl 示例

curl -X POST "https://api.example.com/v1/evaluation/upload" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -F "files=@interview.mp4" \
  -F "files=@photo1.jpg" \
  -F "candidateName=张三"

成功响应

{
  "code": 0,
  "message": "success",
  "data": {
    "taskId": "eval_7a3f2b1c8d4e5f6a",
    "status": "PROCESSING",
    "createdAt": "2026-07-20T10:30:00+08:00"
  },
  "timestamp": 1752979200000
}

常见错误

场景code说明
无认证1001未授权：缺少认证信息
无效 authKey1002未授权：authKey 无效或已过期
未上传文件2004未上传任何文件
文件类型不支持2002支持：视频(.mp4/.mov/.avi)，图片(.jpg/.jpeg/.png/.webp)
文件超大小2003视频最大 500MB，图片最大 20MB

5. 接口二：GET 查询评估结果

请求参数（Query String）

参数名类型必填说明

curl "https://api.example.com/v1/evaluation/result?taskId=eval_7a3f2b1c8d4e5f6a" \

6. 评估结果字段说明

顶层字段

字段类型说明
totalScorefloat总分（满分100）。全部20小项得分直接加总即为百分制
gradestring等级代码：S / A+ / A / B / C / C-
gradeLabelstring等级中文名：顶级人才 / 优秀 / 良好 / 一般 / 较弱 / 不达标
dimensionsDimension[]10个维度评估详情
generalTagsTag[]通用标签（体力体能、语言能力、技能特长等）
industryAdvantagesAdvantage[]行业优势匹配（美妆个护、服饰鞋包等）
overallCommentstring综合评价总结

维度（Dimension）字段

字段类型说明
dimensionIdinteger维度编号 1–10
namestring维度中文名称
weightfloat权重（0.00–1.00）
weightedScorefloat该维度加权得分
maxScorefloat该维度满分
subItemsSubItem[]子项评分列表

子项（SubItem）字段

字段类型说明
namestring子项名称
scoreinteger实际得分
maxScoreinteger满分
commentstring评语

总分计算

全部 20 小项得分直接加总即为百分制总分（满分 100）。

20 项满分分布：前 9 类 15 项共 82 分，第 10 类话术天赋 6 项每项 3 分共 18 分。

等级映射

总分区间等级含义
95–100S顶级人才
88–94A+优秀
80–87A良好
70–79B一般
60–69C较弱
< 60C-不达标

4002003文件大小超出限制
4002004未上传任何文件
4043001评估任务不存在
5005001文件存储失败
